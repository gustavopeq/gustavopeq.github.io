Moviebase Watchlist

Description

Moviebase Watchlist is a Native Android app that consumes a Movie API and show a lists of movies to the user. Besides that, the user can select the movies they want to watch later, and add to a “To Watch” list.

Moviebase Watchlist has been published in the Google Play Store, you can download it here.

Options available for the user:

  • Movies currently playing
  • Upcoming movies
  • Top rated movies
  • Popular movies
  • Search movies by title
  • To Watch list

The app is open source and it can be found on my Github page.


Features

  • REST API
  • Repository
  • Room Database
  • Recycler views
  • Coroutines
  • Fragment navigation
